SPA React
---
 
React APP to navigate beetween two pages, Home and About.
- Router must manage navigation
- Both html page must show user name at top using Redux



Usage
---
 
Start the development server with this command:
 
```
npm start
```
 
 
Setup
---
 
```
npm install
```
 
 
 
Compile
---
 
```
npm run compile
```
